FAQ

(Frequently Asked Questions)

¿Qué es esto?

Hola! Bienvenido a Underwiki, esta es una página creada con el objetivo de brindar una guía a los jugadores sobre las habilidades que se aprenden con el paso de los niveles y poder orientarlos a la hora de elegir que clase jugar.

Esto se creó como una necesidad, ya que en las otras wikis o guías había errores que se iban arrastrando, desde costos de SP hasta daños en las habilidades.

¿Por qué usaría esta wiki y no una de las otras que está más completa?

Esto es un comienzo, como cualquier wiki, tiene una tonelada de información que lentamente se irá agregando con el pasar del tiempo.

Es recomendable usar esta wiki, al jugar Undergames, ya que está basada en la base de datos que se utiliza en el servidor actual y toda la información es correcta.

Mientras tanto se puede utilizar otra base de datos, pero con cuidado, ya que puede tener información incorrecta a la de in-game.

¿Por qué las descripciones de las habilidades son diferentes a las que están en el juego?

Undergames se destaca por la precisión y calidad de sus tooltips. Para preservar la autoría intelectual del trabajo, se utilizaron las descripciones base del juego. Si desea una mejor comprensión del skill, se recomienda visualizarlo directamente in-game.

Encontré un error, ¿Qué hago?

Si es un error de diseño, sobre daño, costo o atributo en las habilidades se agradece que se reporte al Discord para que pueda ser arreglado.

Si resulta no ser un error, los moderadores estarán contentos de poder explicarle el funcionamiento de la habilidad.

En el server que jugué esta habilidad no funcionaba así, ¿Es custom?

Todas las habilidades son 100% retail, sacadas de una base de datos oficial. Normalmente en otros servidores se producen errores en las habilidades y se toman como que están bien, pero originalmente están bug, mal implementadas o modificadas.

Encontré una habilidad que sé que tiene atributo, pero no figura que lo tenga, no tiene ni el marco ni el ícono que lo indique.

Como se mencionó anteriormente, hay errores que se van arrastrando entre wikis, hay habilidades que otorgan atributo (Holy weapon), pero el skill en sí no lo tiene, el efecto o buff que aplica al usuario si lo posee.

Ante cualquier duda o consulta que no se logró resolver en estas preguntas, estás invitado a unirse a Discord para que podamos aclararla.

Disfruten.
